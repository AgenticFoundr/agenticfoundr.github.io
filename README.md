# AgenticFoundr Website

The official website for [AgenticFoundr](https://agenticfoundr.com) — AI agents for solopreneurs.

## About

AgenticFoundr helps solo entrepreneurs build one-person enterprises with AI agents. Our tools, templates, and playbooks give every dreamer a team.

**Your dream deserves a team.** 🤖

## Structure

```
index.html       — Homepage
products.html    — Product catalog (23 digital products)
free.html        — Free downloadable resources
about.html       — Mission, values, and team
blog.html        — Blog (launching soon)
404.html         — Custom 404 page
css/style.css    — Global styles
assets/          — Images and media
CNAME            — Custom domain config
```

## Tech Stack

- Pure HTML + CSS (no frameworks, no build step)
- Hosted on GitHub Pages
- Custom domain: agenticfoundr.com

## Local Development

Just open any `.html` file in your browser. No build step required.

For live reload during development:
```bash
npx serve .
```
